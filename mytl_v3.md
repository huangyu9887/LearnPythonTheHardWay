
# 描述

- 提示输入字符，然后输入 ， ff
- 输出 
  >（u 'ff', 开始时间 timestamp）
- 提示输入字符，然后输入 .
- 输出 
  > done, xx s
- 提示输入字符，然后输入 ， gg
- 输出 
  >（u 'gg', 开始时间 timestamp）
- 提示输入字符，然后输入 .
- 输出 
  > done, xx s
- 提示输入字符，然后输入 ?
- 输出 
  ```
  0:    ff
  1:    gg
  ```
- 提示输入字符，然后输入要替换的元素 "> 0  ssss"
- 输出
  ```
  0:   ssss
  1:   gg
  ```
- 提示输入字符，然后输入"/"
- 输出 
  ```
  总计 xx s
  0:   ssss
  1:   gg
  ```
